# Drug Shortage Prediction Dataset

## Deskripsi
Dataset ini digunakan untuk memprediksi kekurangan obat menggunakan metode Machine Learning.

## Sumber Data
Data diperoleh dari:
- Rumah Sakit
- Sistem logistik obat

## Struktur Data
| Kolom | Deskripsi |
|------|----------|
| nama_obat | Nama obat |
| stok | Jumlah stok |
| penggunaan | Pemakaian |
| status | 0 = aman, 1 = shortage |

## Tujuan
Untuk membantu early warning system dalam pengadaan obat.

## Metode
Model yang digunakan:
- Random Forest
